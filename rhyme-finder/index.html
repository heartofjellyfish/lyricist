How to read this — Pattison's cheat sheet

Every rhyme decision is a feeling decision. Stable rhyme supports stable ideas. Unstable rhyme supports unstable ideas.

  • Stability 5 · Perfect — same vowel, same coda, different onset. Slams the gate. Watch for clichés.
  • Stability 4 · Family — coda consonants from the same phonetic family. Partners (b↔p) feel tighter than companions (b↔d↔g) for plosives; reverse for fricatives.
  • Stability 3 · Additive / Subtractive — one side has an extra consonant. r and l hide added sounds best.
  • Stability 2 · Assonance — same vowel, unrelated codas. Rings but hangs. Feminine assonance is strong; masculine weak.
  • Stability 1 · Consonance — same coda, different vowel. Suspended, unresolved. "If the line allows 'Alas!', use consonance."

Masculine ≠ Feminine pairings are flagged — Pattison's iron law, broken only on purpose (partial rhyme).
