How to read this — Pattison's cheat sheet

Every rhyme decision is a feeling decision. Stable rhyme supports stable ideas. Unstable rhyme supports unstable ideas.

  • Perfect · same vowel + same coda. Slams the gate. Watch for clichés.
  • Family · same vowel + coda from the same phonetic family. Partners (b↔p) feel tighter than companions (b↔d↔g) for plosives.
  • Additive / Subtractive · one side has an extra coda consonant. r and l hide best.
  • Assonance · same vowel, unrelated coda. Rings but hangs. Feminine is stronger than masculine.
  • Consonance · same coda, different vowel. Suspended, unresolved. "If the line allows 'Alas!', use consonance."

⚠︎ flags a stress-class mismatch (masculine vs feminine). 🚨 flags a Pattison cliché (love/above, heart/apart, …).
